Terrain Generator

LINK

Details

This small program creates an n-by-n meter terrain by using perlin noise. It uses a few external files for the textures, but, this helps it create visually beautiful images.

No. of squares in model:$$n^2 + 1$$ No. of triangles in model:$$2n^2 + 2$$
